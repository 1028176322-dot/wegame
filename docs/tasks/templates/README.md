# docs/tasks/templates/

## Purpose
方案 §21 最小模板集，供新建任务/决策/回执时复制使用。

## Owner
Coordinator

## 模板清单

| 文件 | 用途 | 方案 |
|---|---|---|
| `TASK_CARD.md` | 任务卡 | §21.1 |
| `TASK_STATE.json` | 任务机器状态 | §21.2 |
| `CHANGE_LIST.md` | 冻结变更清单 | §21.6 |
| `CHECKPOINT.json` | 检查点 | §21.7 |
| `HANDOFF.md` | 交接/恢复 | §21.8 |
| `REVIEW_FINDINGS.md` | 独立审查 | §21.9 |
| `ADR.md` | 决策记录 | §21.10 |
| `RCA.md` | 根因分析 | §21.12 |
| `RELEASE_CHECKLIST.md` | 发布检查表 | §21.13 |

## 用法
复制模板到目标位置并重命名为带任务ID的文件（如 `TASK-042_CHANGE_LIST.md`），不要直接修改本目录模板。
